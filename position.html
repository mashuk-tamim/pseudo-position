CSS position

  1. Static: এটি default position.
  2. Relative: নিজের জায়গা ছাড়ে না অন্য element ফাঁকা জায়গা দখল করবে। কিন্তু আবার এটাকে নিজের কারেন্ট পজিশনের relative এ সরানো যায়। সেক্ষেত্রে নিজের জায়গা ফাঁকা রেখে সরবে।
  3. Absolute: নিজের জায়গা হারাবে এবং parent এর পজিশন অনুযায়ী সরবে নড়বে। parent না থাকলে root এর পজিশন অনুযায়ী সরবে নড়বে।
  4. Sticky: position:sticky দিয়ে top,right,bottom,left value দিলে সেটা সেখানে স্থির হয়ে যাবে। মাউস স্ক্রল করলেও থেকে যাবে। তবে parent এর এরিয়া শেষ হয়ে গেলে বিদায় নেবে।
  5. Fixed: sticky এর মতই তবে পার্থক্য শুধু parent এর উপর নির্ভর করবে না, parent শেষ হয়ে গেলেও থাকবে অর্থাৎ root এর সাপেক্ষে fixed হয়ে যাবে।
